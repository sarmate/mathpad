Text, structure and inline formatting Headings and paragraphs

mathpad organises a document into chapters, sections and subsections, all numbered automatically. Plain paragraphs use ordinary <p> tags, and inline mathematics is written between dollar signs, e.g. $a^2 + b^2 = c^2$ or $\displaystyle \lim_{n \to \infty} \frac{1}{n} = 0$.

A subsection

Subsections nest under sections and inherit the numbering (1.1.1, and so on). Lists work as usual:

Formal blocks Definitions, theorems and friends A function $f$ is differentiable at $a$ when the limit $$f'(a) = \lim_{h \to 0} \frac{f(a+h) - f(a)}{h}$$ exists and is finite. In a right triangle with legs $a$, $b$ and hypotenuse $c$, $$a^2 + b^2 = c^2.$$ The ROC badge marks a result worth memorising. For any constants $\alpha, \beta$, $\displaystyle \int (\alpha f + \beta g) = \alpha \int f + \beta \int g$. Every differentiable function is continuous. Triangle inequality For all real $x, y$, $\; |x + y| \le |x| + |y|$. Every monotonic bounded sequence converges. Each formal block is numbered automatically and carries its own colour and icon. Finding an affine function from two points
  1. Compute the slope $a = \dfrac{y_2 - y_1}{x_2 - x_1}$.
  2. Substitute into $y = ax + b$ to find $b$.
 Do not confuse the value $f(a)$ with the derivative $f'(a)$. We prove the slope of an affine function is constant. Take two points $M(x_1, y_1)$ and $N(x_2, y_2)$ on the line $y = ax + b$: $y_2 - y_1$ & $=$ & $(a x_2 + b) - (a x_1 + b)$ & definition & $=$ & $a(x_2 - x_1)$ so the slope $\dfrac{y_2 - y_1}{x_2 - x_1} = a$ does not depend on the chosen points.

 Pedagogical blocks Examples, activities and exercises Arithmetic sequence The sequence defined by $u_n = 2n + 1$ is arithmetic with common difference $2$: $\;1, 3, 5, 7, \dots$ Warm-up For $f(x) = 3x - 2$, compute $f(0)$, $f(1)$ and $f(2)$, then plot the three points. Determining an affine function An affine function $f$ satisfies $f(2) = 5$ and $f(-1) = -1$. Find $f(x)$, then compute $f(10)$. Slope: $a = \dfrac{5 - (-1)}{2 - (-1)} = 2$. From $f(2) = 4 + b = 5$ we get $b = 1$, so $f(x) = 2x + 1$ and $f(10) = 21$.

Drag the slider to change the number of rectangles approximating the area under the curve $f(x) = \tfrac{1}{2}x(5 - x)$ on $[0, 5]$ (a Riemann sum), and watch the estimate converge:

var f = function (x) { return 0.5 * x * (5 - x); }; var n = board.create('slider', [[0.2, 4.0], [1.9, 4.0], [1, 8, 40]], { name: 'n', snapWidth: 1 }); board.create('functiongraph', [f, 0, 5], { strokeColor: '#be185d', strokeWidth: 2.5 }); board.create('riemannsum', [f, function () { return n.Value(); }, 'middle', 0, 5], { fillColor: '#0891b2', fillOpacity: 0.3 }); board.create('text', [3.0, 4.0, function () { return 'area ≈ ' + JXG.Math.Numerics.riemannsum(f, n.Value(), 'middle', 0, 5).toFixed(3); }]); Code and data Runnable Python
